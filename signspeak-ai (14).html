From webcam to spoken words — the complete pipeline powering real-time ASL translation in your browser.

01
📷
Webcam / Video Capture
Your browser captures live video via WebRTC getUserMedia at 30+ FPS. You can also upload a video file for offline analysis — both routes feed into the same MediaPipe pipeline.
WebRTC · File API
02
🖐️
MediaPipe Hand Detection
Each frame is passed to Google's MediaPipe Hands — a WASM-compiled neural network that detects up to 2 hands and maps 21 3D landmarks per hand in real time, entirely on-device.
MediaPipe · WASM · 21 points × 2 hands
03
📐
Normalization
Landmarks are centered on the wrist and scaled by wrist-to-middle-MCP distance — producing 63 scale- and position-invariant features that work at any camera distance.
63 features · Scale-invariant
04
🧠
TensorFlow.js Classifier
A 3-layer dense network (63→128 ReLU→64 ReLU→N softmax) trained in the browser on your own samples. Top-3 predictions shown live with confidence. Falls back to geometric rules until trained.
TF.js · In-browser training · Softmax
05
⏳
Hold-Frame Confirmation
A sign is confirmed only after being predicted consistently for 8 consecutive frames — preventing false positives from transient hand positions between signs.
8-frame threshold · Debounce
06
✦
Trigram Auto-Complete
A trigram model predicts the 6 most likely next words from your last 2 signed words. Covers common ASL conversational patterns — click any chip to insert instantly.
Trigram NLP · Click-to-insert
07
🤖
AI Sentence Builder
Signed words are sent to a language model that reconstructs them into natural English — handling both spelled letters and word-level input.
LLM · Sentence reconstruction
08
🔊
Text-to-Speech
Every confirmed word and AI sentence is spoken aloud via the Web Speech API — multiple voices, adjustable speed, no plugin needed.
Web Speech API · Multi-voice

About This Project
BRIDGE THE GAP

SignSpeak AI is an assistive technology project that uses computer vision and deep learning to make real-time ASL translation accessible to everyone — with zero hardware, no internet required, and no data leaving your device.

Built for education, healthcare, public services, and everyday communication — empowering deaf and hard-of-hearing individuals with a browser-based translation tool backed by a trainable neural network.
